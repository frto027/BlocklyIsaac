如何编辑文档

文档和编辑器源代码在一起。

文档目录（改这里）：
bi-docs/bi-docs-cn/docs

文档构建目录（自动生成的，不要动它）：
bi-docs/bi-docs-cn/site

文档通过mkdocs自动生成。

在bi-docs下使用mkdocs serve、mkdocs build等指令来构建文档，详见mkdocs。

用MkDocs构建，使用Read the Docs提供的theme。
