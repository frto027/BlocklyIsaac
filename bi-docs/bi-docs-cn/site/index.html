简介


此文档简介以撒的积木系统的基本用法。

以撒的积木是一个基于Google Blockly开发的图形化脚本编辑工具。它允许用户以“搭积木”的方式来生成游戏的mod。在达到lua编程的几乎全部能力的情况下，不要求具备任何的lua编程经验，降低了游戏的编程门槛。

此工程为开源工程，已经发布在gitee和 github上，如果发现bug，欢迎进行反馈。
